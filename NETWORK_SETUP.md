# Развёртывание CartridgeApp в локальной сети

## Схема работы

```
       СЕРВЕР                        ПОЛЬЗОВАТЕЛИ
  ┌─────────────────┐          ┌──────────────────────┐
  │  cartridge_app\ │          │  Любой ПК в сети     │
  │  ├── app.py     │   ЛВС    │  Открыть браузер:    │
  │  ├── data.db    │ ◄──────► │  http://192.168.X.X  │
  │  └── backups\   │          │       :5000           │
  └─────────────────┘          └──────────────────────┘
  Python + Flask                Chrome / Edge / Firefox
  Работает постоянно            Ничего не устанавливать
```

---

## ШАГ 1 — Настройка сервера (один раз)

### 1.1 Установить Python на сервере
Скачать: https://python.org/downloads
При установке отметить: **[x] Add Python to PATH**

### 1.2 Скопировать папку cartridge_app на сервер
Например: `C:\CartridgeApp\`

### 1.3 Запустить сервер
Запустить файл **`start.bat`**

Окно покажет, что сервер запущен. Узнайте IP-адрес сервера:
```
ipconfig
```
Найдите строку **IPv4-адрес**, например `192.168.1.100`

### 1.4 Разрешить порт в брандмауэре Windows
Открыть CMD от имени Администратора и выполнить:
```
netsh advfirewall firewall add rule name="CartridgeApp" dir=in action=allow protocol=TCP localport=5000
```

### 1.5 Автозапуск при старте Windows (необязательно)
Чтобы сервер запускался автоматически при включении компьютера, создайте задачу в Планировщике задач Windows:
- Действие: запустить `python app.py` из папки приложения
- Триггер: при запуске системы
- Выполнять от имени: учётная запись с нужными правами

## ШАГ 2 — Настройка у пользователей (на каждом ПК)

### Вариант А — просто открыть браузер
1. Открыть Chrome / Edge / Firefox
2. В адресной строке написать: `http://192.168.1.100:5000`
   (вместо 192.168.1.100 — IP вашего сервера)
3. Ввести логин и пароль

### Вариант Б — создать ярлык на рабочем столе
1. Создать новый текстовый файл `CartridgeApp.url` с содержимым:
```ini
[InternetShortcut]
URL=http://192.168.1.100:5000
```
2. Скопировать файл на рабочий стол пользователей

## Как узнать IP сервера

На сервере открыть CMD и выполнить:
```
ipconfig
```
Найти строку **IPv4-адрес** — например `192.168.1.100`

## Пользователи и права

| Роль     | Что может делать                              |
|----------|-----------------------------------------------|
| admin    | Всё: просмотр, ввод, управление, бэкапы       |
| operator | Просмотр и ввод данных о расходе              |
| viewer   | Только просмотр, ничего не меняет             |

Добавить пользователей: войти как admin → раздел **Администрирование**

| Файл        | Назначение                                     |
